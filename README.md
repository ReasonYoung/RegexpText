# RegexpText
正则表达式，一个十分古老而又强大的文本处理工具，仅仅用一段非常简短的表达式语句，便能够快速实现一个非常复杂的业务逻辑。熟练地掌握正则表达式的话，能够使你的开发效率得到极大的提升。
Demo中提供了手机号码、密码格式、邮箱格式等正则表达式的使用（以实现），在Utils类中也有用户姓名、用户身份证号码、员工号码、正则匹配URL等相关的具体代码，可直接在项目中复制调用；
